> 一个面向 AI 图像生成的轻量级在线工具。  
> 库中展示、示例与相关内容均源自 AI 生成。

## 在线体验

访问地址：<https://lochou-grsai-banana-985370071424.asia-south1.run.app/>

## 项目说明

Lochou AI Studio 主要用于在线 AI 生图，支持调用部分主流 Banana 模型以及 GPT Image 2 模型。当前版本聚焦于图像生成能力，没有接入聊天功能。

## 功能特性

- 支持部分主流 Banana 模型调用（非全部模型）
- 支持 GPT Image 2 模型
- 主要用于 AI 图像生成
- 支持多图并发生成
- 支持画面比例 / 尺寸选择
- 支持上传参考图
- 支持预览历史任务结果图
- 支持 API Key 与 API Endpoint 配置

## 使用方式

1. 打开在线体验地址。
2. 选择需要使用的模型，例如 Banana 或 GPT Image 2。
3. 输入图像描述词。
4. 根据需要上传参考图。
5. 选择合适的画面比例和分辨率。
6. 点击生成，等待任务完成。
7. 在历史任务区域预览生成结果。

## 注意事项

- 本项目主要用于生图，不包含聊天功能。
- Banana 模型仅支持部分主流模型，并非完整模型列表。
- 生成结果、示例图片及库中内容均源自 AI 生成。
- 使用前请确保 API Key 与 API Endpoint 配置正确。
